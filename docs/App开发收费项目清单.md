# App 开发收费项目清单

> 面向中国大陆 App 开发、上架和运营。费用会随平台、服务商、政策和业务类型变化，最终以各平台付款页、合同或官方审核要求为准。

## 固定平台费用

| 项目 | 是否常见 | 参考费用 | 说明 |
| --- | --- | --- | --- |
| Apple Developer Program | iOS 上架必需 | 99 美元/年；中国大陆通过 Apple Developer App 注册时通常为 688 元/年 | 用于 App Store 上架、TestFlight、证书和描述文件 |
| Apple Developer Enterprise Program | 仅企业内部分发 | 299 美元/年 | 不用于普通 App Store 上架，审核和使用限制更严格 |
| Google Play Console 开发者账号 | 上架 Google Play 必需 | 25 美元一次性 | 需要完成身份验证；国内应用不一定必须上 Google Play |
| 国内安卓应用市场账号 | 常见 | 多数免费 | 华为、小米、OPPO、vivo、应用宝等通常不收固定上架费，但会有企业认证、资质和合规成本 |
| D-U-N-S 编号 | iOS 公司账号必需 | 通常免费 | 组织身份申请 Apple Developer Program 时需要，详见 [邓白氏编码申请及费用](./邓白氏编码申请及费用.md) |

## 设备和开发环境

| 项目 | 费用类型 | 说明 |
| --- | --- | --- |
| Mac 电脑 | 一次性硬件成本 | iOS 证书、Xcode 打包、真机调试通常需要 Mac；也可使用云 Mac 或 CI 服务 |
| iPhone/iPad 测试机 | 一次性硬件成本 | 用于真机调试、适配、审核问题复现 |
| Android 测试机 | 一次性硬件成本 | 国内安卓机型差异较大，建议至少覆盖主流品牌和系统版本 |
| 云构建/CI/CD | 按量或订阅 | EAS、GitHub Actions、Codemagic、Bitrise、云 Mac 等可能产生费用 |

## 资质和合规费用

| 项目 | 是否常见 | 说明 |
| --- | --- | --- |
| ICP 备案 | 国内服务器/域名常见 | 备案本身通常免费，但需要购买符合备案要求的国内云服务器、域名等资源 |
| App 备案 | 国内上架和分发常见 | 工信部要求在中国境内从事互联网信息服务的 App 履行备案手续；备案本身通常不收费 |
| 公安联网备案 | 视业务和地区要求 | 网站或 App 涉及互联网信息服务时可能需要办理 |
| 软件著作权 | 国内安卓市场常见 | 部分应用市场、游戏、金融、教育等类目可能要求；自行申请成本低，代理通常收费 |
| 隐私合规检测 | 常见 | 国内应用市场可能要求按隐私政策、权限、SDK 合规规则整改，第三方检测可能收费 |
| 特殊行业资质 | 视业务而定 | 新闻、出版、教育、医疗、金融、直播、音视频、游戏等可能需要额外许可证或审批 |
| 法务/隐私政策/用户协议 | 建议准备 | 可自行编写，但商业项目建议让法务审核 |

## 云服务和基础设施

| 项目 | 收费方式 | 说明 |
| --- | --- | --- |
| 域名 | 年费 | 官网、接口域名、隐私政策页面通常需要 |
| SSL 证书 | 免费或年费 | Let's Encrypt 等可免费；企业级或通配符证书可能收费 |
| 云服务器/容器/函数计算 | 按量或包年包月 | 后端接口、管理后台、定时任务等 |
| 数据库 | 按规格和容量 | MySQL、PostgreSQL、MongoDB、Redis 等 |
| 对象存储 | 按存储量和流量 | 图片、视频、附件、安装包、日志等 |
| CDN | 按流量 | 静态资源、图片、视频分发常用 |
| 日志、监控、告警 | 免费额度或订阅 | 崩溃、性能、错误日志、业务监控 |
| 备份和灾备 | 按容量 | 数据库备份、对象存储备份、跨区容灾 |

## 第三方能力费用

| 能力 | 收费方式 | 常见场景 |
| --- | --- | --- |
| 短信验证码 | 按条计费 | 登录、注册、找回密码 |
| 邮件服务 | 免费额度或按量 | 验证邮件、通知邮件 |
| [推送服务](./App推送通道申请及费用.md) | 免费、按量或套餐 | APNs、华为、小米、vivo、OPPO、Firebase、聚合推送等 |
| 地图和定位 | 免费额度或按量 | 门店、轨迹、打车、外卖、位置检索 |
| 支付服务 | 交易手续费 | 微信支付、支付宝、银行卡支付等 |
| 即时通讯 IM | 按 MAU、消息量或套餐 | 聊天、客服、群组 |
| 音视频/直播 | 按分钟、带宽或套餐 | 会议、直播、语音房 |
| OCR/人脸识别/实名认证 | 按次计费 | 身份核验、票据识别、实名制 |
| AI 接口 | 按 token、调用量或套餐 | 智能客服、内容生成、审核、翻译 |
| 内容安全审核 | 按量或套餐 | 文本、图片、音频、视频审核 |
| 统计分析/归因 | 免费或订阅 | 用户行为、渠道投放、留存分析 |

## 应用市场和交易抽成

| 场景 | 可能费用 | 说明 |
| --- | --- | --- |
| iOS 数字内容/会员/虚拟商品 | Apple IAP 抽成 | 通常需要使用 Apple 内购，费率取决于开发者计划、订阅状态和地区规则 |
| Google Play 数字内容/会员/虚拟商品 | Google Play 服务费 | Google 官方服务费规则会按收入、订阅、地区和计划变化 |
| 实物商品和线下服务 | 支付通道手续费 | 一般使用微信/支付宝/银行卡等支付通道，不走 Apple IAP |
| 广告变现 | 平台分成 | 穿山甲、优量汇、AdMob 等广告平台按规则分成 |
| 国内应用市场推广 | 预算可选 | 搜索广告、首发、品牌专区、买量投放等 |

## 设计、测试和发布物料

| 项目 | 说明 |
| --- | --- |
| App 图标、启动图、截图 | App Store 和安卓市场都需要，可能产生设计成本 |
| 隐私政策页面、官网、客服入口 | 上架审核常用资料 |
| 测试服务 | 兼容性测试、安全测试、渗透测试、众测等可能收费 |
| 软著、资质、备案材料整理 | 找代理会产生服务费 |
| 多语言翻译 | 海外上架或多地区运营时常见 |

## MVP 预算判断

如果只是做一个普通工具类 App，并且只先上架 iOS：

- 必需费用通常是 Apple Developer Program 年费。
- 公司账号需要 D-U-N-S 编号，但通常免费。
- 如果后端使用国内云服务，通常还会有域名、服务器、备案相关成本。
- 如果需要短信登录、文件上传、地图、支付、AI 等能力，对应服务会产生持续费用。

如果还要上架 Google Play：

- 需要 Google Play Console 一次性 25 美元注册费。
- 如果 App 销售数字内容、会员或虚拟商品，还要考虑平台服务费。

如果要在国内安卓市场分发：

- 账号本身多数不收上架费。
- 更常见的成本是企业认证、软著、App 备案、隐私合规整改、应用加固和各市场材料准备。
