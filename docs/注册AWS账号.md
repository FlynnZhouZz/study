# 注册AWS账号

[AWS](https://aws.amazon.com/cn)

## 创建账号

### 步骤一: 填写账户信息
- 输入邮箱地址（建议使用常用邮箱）
- 设置AWS账户名称（可后续修改）
- 验证电子邮件（会给你邮箱发送验证码）
- 创建密码（需包含大小写字母、数字和特殊字符）

### 步骤二: 选择账户计划

- 免费计划: 6个月免费
- 付费计划: 即用即付定价模式计费

### 步骤三: 联系信息

- 选择"个人"或"企业"账户
- 填写详细联系信息（姓名、地址、电话、邮政编码等）
- 确保信息真实准确

### 步骤四: 账单信息
- 输入信用卡信息（用于验证，不会立即扣费）
- 首次验证会扣除1美元（或等值当地货币），通常24小时内退还

### 步骤五: 组织信息

- 输入公司信息（组织统一社会信用代码、营业执照照片、联系人信息）

### 审核

审核时间: 3-5工作日

## 开启MFA多因素认证

可自行选择认证方式：设备密钥、APP授权等
