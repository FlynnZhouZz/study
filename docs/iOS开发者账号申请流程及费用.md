# iOS 开发者账号申请流程及费用

> 面向中国大陆个人或公司申请 Apple Developer Program。费用和付款方式可能会调整，最终以 Apple 注册付款页展示为准。

## 账号类型

| 类型 | 适合场景 | App Store 开发商名称 | 关键要求 |
| --- | --- | --- | --- |
| 个人账号 | 个人开发者、个人项目、独资企业/一人公司 | 个人法定姓名 | Apple 账户开启双重认证，完成身份验证 |
| 组织账号 | 公司主体上架 App、需要多人协作 | 公司法人实体名称 | D-U-N-S 编号、公司域名邮箱、公开网站、法律约束授权 |
| 企业账号 | 仅面向公司内部员工私密分发 App | 公司法人实体名称 | Apple Developer Enterprise Program，审核更严格，不用于 App Store 上架 |

一般 App Store 上架选择 Apple Developer Program 即可；只有需要通过内部系统或 MDM 向员工私密分发时，才考虑 Enterprise Program。

## 费用

| 项目 | 费用 | 说明 |
| --- | --- | --- |
| Apple 账户 | 免费 | 登录并接受 Apple Developer Agreement 后可免费使用基础开发能力 |
| Apple Developer Program | 99 美元/年 | 价格可能因地区不同；中国大陆通过 Apple Developer App 注册时显示为 688 元/年，实际以付款页为准 |
| Apple Developer Enterprise Program | 299 美元/年 | 用于企业内部私密分发，不适合普通 App Store 上架 |
| D-U-N-S 编号 | 通常免费 | 公司/教育机构申请组织账号必须提供，个人账号不需要 |
| 资料公证/翻译 | 视情况产生 | Apple 可能要求补充经公证的商业文件，费用取决于办理渠道 |

非营利组织、取得认可的教育机构或政府机构满足条件时，可以申请 Apple Developer Program 会费豁免。

发票：如果通过中国大陆 Apple Store Online 完成购买并需要发票，可以按 Apple 官方说明联系 400-666-8800 咨询。

## 申请前准备

### 个人账号

- 一个 Apple 账户，并开启双重认证。
- Apple 账户中的姓名、地址、电话保持真实且最新。
- 中国大陆账户持有人需要通过 Apple Developer App 完成身份验证。
- 准备身份证号、手机号，并按 App 提示拍摄自拍照。
- 用于付款的银行卡、信用卡或付款页支持的其他方式。Apple Developer Program 会员费用不接受 Apple 账户余额付款。

### 组织账号

- 一个 Apple 账户，并开启双重认证，建议使用公司域名邮箱。
- 公司营业执照信息，确保法人实体名称与注册信息一致。
- D-U-N-S 编号。没有编号时先参考 [邓白氏编码申请及费用](./邓白氏编码申请及费用.md) 申请。
- 公司官网：必须公开可访问，域名要能体现与公司有关联，不建议使用社交媒体页或空壳页。
- 公司域名邮箱，例如 `name@company.com`。
- 法律约束授权：申请人需要是所有者/创始人、高管、高级项目负责人，或有高级员工授权。
- 法律联系人：如果申请人不是所有者/创始人，Apple 可能要求提供可确认授权的联系人。
- 可能需要补充盖章营业执照、授权证明、经过公证的商业文件等材料。

## 个人账号申请流程

1. 安装并打开 Apple Developer App。
2. 使用准备好的 Apple 账户登录，并确认已开启双重认证。
3. 在“账户”中选择注册或验证身份。
4. 按提示填写法定姓名、身份证号、手机号等信息。
5. 完成自拍照验证，提交审核。
6. 审核通过后，同意 Apple Developer Program 许可协议。
7. 完成年费支付，等待会员资格激活。
8. 激活后登录 [Apple Developer](https://developer.apple.com/account/) 和 [App Store Connect](https://appstoreconnect.apple.com/) 检查账号状态。

## 组织账号申请流程

1. 确认公司是否已有 D-U-N-S 编号。
   - 如果没有，先申请 D-U-N-S 编号。
   - 收到编号后，通常需要等待 D&B 将信息同步给 Apple 后再注册。
2. 打开 [Apple Developer Program 注册页](https://developer.apple.com/cn/programs/enroll/)。
3. 使用申请人的 Apple 账户登录。
4. 选择以组织身份注册。
5. 填写申请人身份信息，并完成必要的身份验证。
6. 填写组织信息：
   - 法人实体名称
   - D-U-N-S 编号
   - 公司网站
   - 公司域名邮箱
   - 公司电话和地址
   - 申请人的职位和法律约束授权信息
7. 提交后等待 Apple 审核。
8. 如果 Apple 电话或邮件核验，确保申请人、法律联系人、公司总机或行政人员能及时响应。
9. 审核通过后，同意 Apple Developer Program 许可协议。
10. 完成年费支付，等待会员资格激活。
11. 激活后在 Apple Developer 后台邀请团队成员，并在 App Store Connect 配置应用上架信息。

## 时间预估

| 阶段 | 参考时间 |
| --- | --- |
| D-U-N-S 查询/申请 | 已有编号可直接使用；新申请通常最多 5 个工作日收到编号 |
| D-U-N-S 同步到 Apple | 收到编号后通常最多 2 个工作日 |
| Apple 身份/组织审核 | 常见为数天，资料不一致或需补件时会更久 |
| 付款后激活 | 通常 24 小时内收到确认，超时可带 Enrollment ID 联系支持 |

## 国内常见注意事项

- 个人账号的开发商名称会显示个人法定姓名，不能显示公司名。
- 公司主体上架建议申请组织账号，否则后续品牌、合同、收款和团队协作都容易受影响。
- 组织账号的公司名称要使用法人实体名称，不接受经营别称、商号、分公司等。
- Apple 账户里的姓、名不要写昵称、英文名或公司名，否则可能导致审核延迟。
- D-U-N-S 信息要与营业执照、官网、公司英文名/地址尽量一致。
- 公司网站要提前准备好，不要只放备案页、空页面或社媒链接。
- 申请期间留意邮件和电话，尤其是 Apple、D&B、公司域名邮箱收到的信息。
- 通过 Apple Developer App 注册的会员资格通常是自动续期订阅；不想续费时要提前取消。
- 会员过期会影响证书、描述文件、TestFlight 和 App 更新发布，生产账号建议设置续费提醒。
