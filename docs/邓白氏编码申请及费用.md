# 邓白氏编码申请及费用

> 面向中国大陆公司申请 Apple Developer Program 组织账号时使用。D-U-N-S 编号由 Dun & Bradstreet（D&B，邓白氏）分配和维护，Apple 用它来验证组织身份和法人实体状态。

## 什么是 D-U-N-S 编号

D-U-N-S 编号是一组唯一的 9 位数字，用于识别企业实体的具体地点。公司、教育机构以组织身份注册 Apple Developer Program 或 Apple Developer Enterprise Program 时需要提供；政府组织可选；个人账号不需要。

Apple 只接受法人实体以组织身份注册，不接受经营别称、虚构公司、商号或分公司。如果主体是个体工商户、独资企业/一人公司，可能需要按个人身份注册。

## 费用

| 项目 | 费用 | 说明 |
| --- | --- | --- |
| 查询已有 D-U-N-S 编号 | 免费 | 通过 Apple 的 D-U-N-S 查询入口检索公司是否已有编号 |
| 通过 Apple 入口申请 D-U-N-S 编号 | 免费 | Apple 官方说明为可向 D&B 提交信息获取免费的 D-U-N-S 编号 |
| D-U-N-S 同步到 Apple | 免费 | 收到编号后，D&B 通常最多需要 2 个工作日同步给 Apple |
| 第三方代办/商业报告/认证服务 | 视服务商报价 | 不是注册 Apple Developer Program 的必要费用，谨慎购买 |
| 资料翻译、公证、补充证明 | 视情况产生 | D&B 或 Apple 要求补件时可能产生 |

结论：仅为申请 Apple Developer Program 组织账号，一般不需要向第三方购买“加急”或“认证”服务。Apple 官方也说明，加急 D-U-N-S 创建流程不会缩短等待时间。

## 申请前准备

- 公司营业执照扫描件或照片，信息清晰可读。
- 公司中文全称、英文全称。
- 公司注册地址、办公地址，建议准备英文地址。
- 统一社会信用代码。
- 公司官网，最好是公司自有域名且可公开访问。
- 公司电话、联系人姓名、职位、手机号、邮箱。
- 申请人工作邮箱，建议使用公司域名邮箱，例如 `name@company.com`。
- 公司业务类型、员工人数、成立时间等基础信息。
- 公章。国内企业补充资料时，D&B 可能要求营业执照复印件加盖公章。

## 申请流程

### 1. 先查询是否已有编号

打开 Apple 的 [D-U-N-S 查询入口](https://developer.apple.com/enroll/duns-lookup/)，使用 Apple 账户登录。

按页面要求填写组织信息，建议使用英文信息：

- Legal Entity Name：公司英文法人实体名称
- Headquarters Address：总部/注册地址
- Mailing Address：邮寄地址
- Work Contact Information：申请人的工作联系信息

如果查询到匹配公司，Apple 会通过邮件发送已有 D-U-N-S 编号。拿到编号后，不要急着注册 Apple Developer Program，先确认名称和地址是否与营业执照、官网、公司邮箱信息一致。

### 2. 没有查询到时提交申请

如果 Apple 查询页没有找到你的公司，页面会提供向 D&B 提交信息的入口。按提示填写公司信息后提交。

提交后通常会收到 D&B 的确认邮件，邮件里可能包含 Tracking ID、Case Number 等信息。后续沟通请保留这些编号，方便查询进度。

### 3. 按 D&B 邮件补充材料

D&B 可能通过邮件要求补充资料，常见材料包括：

- 营业执照扫描件或加盖公章的复印件。
- 公司中英文名称、注册地址、办公地址。
- 联系人姓名、职位、电话、邮箱。
- 公司业务类型、员工人数、成立日期。
- 官网或能证明公司正常经营的公开信息。

收到补件邮件后尽快回复。国内申请时，公司英文名称和英文地址要保持前后一致，尽量不要在 Apple、D&B、营业执照翻译、官网里写出多个版本。

### 4. 等待编号生成

Apple 官方说明：申请 D-U-N-S 编号后，最多需要 5 个工作日从 D&B 收到编号。如果超过两周仍未收到，可以联系 D&B 支持。

收到编号后，D&B 还需要最多 2 个工作日把信息同步给 Apple。同步完成后，再用该编号注册 Apple Developer Program 组织账号。

### 5. 在 Apple 注册组织账号

同步完成后，打开 [Apple Developer Program 注册页](https://developer.apple.com/cn/programs/enroll/)，以组织身份注册并填写：

- 法人实体名称
- D-U-N-S 编号
- 公司网站
- 公司域名邮箱
- 申请人的职位和法律约束授权信息

如果 Apple 提示“贵组织不是法人实体”或无法匹配信息，先检查 D-U-N-S 档案中的主体类型、公司名称、地址是否正确，再联系 D&B 更新档案。

## 时间预估

| 阶段 | 参考时间 |
| --- | --- |
| 查询已有编号 | 查询后按邮件结果为准，已有编号通常更快 |
| 新申请 D-U-N-S 编号 | Apple 官方说明最多 5 个工作日 |
| 同步到 Apple | 收到编号后最多 2 个工作日 |
| 超时处理 | 超过两周未收到编号，再联系 D&B 支持 |

实际耗时受资料完整度、D&B 补件、国内节假日、公司电话/邮箱是否能及时响应影响。

## 国内注意事项

- 申请 Apple 组织账号时，D-U-N-S 主体必须是公司法人实体，不能是品牌名、门店名、分公司或简称。
- 公司名称、统一社会信用代码、地址、电话、官网、邮箱域名要尽量一致。
- 没有公司官网时，建议先准备一个基础官网；Apple 不接受社交媒体页、内容极少的页面或域名注册商占位页作为组织网站。
- 联系电话要能接通，行政、前台或联系人要知道这件事，避免 Apple 或 D&B 核验时无人确认。
- 不建议反复提交多个 D-U-N-S 申请，容易造成重复档案或审核混乱。
- 收到 D-U-N-S 编号后不要马上注册 Apple 组织账号，先等同步窗口结束。
- 如果 D&B 档案中的公司信息有误，应联系 D&B 更新，不要在 Apple 注册页硬填一个不一致的信息。
- 仅申请 Apple 开发者组织账号时，通常不需要购买 D&B 商业信用报告、认证标识或第三方代办服务。

## 常见问题

### 个人开发者需要申请邓白氏编码吗

不需要。只有公司、教育机构等组织身份注册时才需要；个人账号不需要 D-U-N-S 编号。

### 有营业执照就一定能申请组织账号吗

不一定。Apple 要求组织是可签署法律协议的法人实体，并且 D&B 档案能验证该状态。个体工商户、独资企业/一人公司等主体可能会被要求以个人身份注册。

### 编号拿到了，Apple 还是查不到怎么办

先等待最多 2 个工作日完成同步。如果仍查不到，检查公司名称、地址、D-U-N-S 编号是否完全一致；确认无误后联系 Apple Developer Support 或 D&B 支持。

### 能不能付费加急

不建议。Apple 官方说明，加急 D-U-N-S 创建流程不会缩短 Apple 这边的等待时间。第三方所谓“加急”不是 Apple Developer Program 注册的必要步骤。
