# HTML标题与段落

## 标题 `<h1>` - `<h6>`

HTML 提供了六个级别的标题标签，从 <h1> 到 <h6>，其中 <h1> 级别最高，<h6> 级别最低。

```html
<h1>这是一级标题</h1>
<h2>这是二级标题</h2>
<h3>这是三级标题</h3>
<h4>这是四级标题</h4>
<h5>这是五级标题</h5>
<h6>这是六级标题</h6>
```

### 标题的重要性

1. 语义化结构：为内容创建清晰的层次结构
2. SEO 优化：搜索引擎通过标题理解页面内容结构
3. 可访问性：屏幕阅读器用户可以通过标题快速导航
4. 用户体验：让读者更容易扫描和理解内容

### 最佳实践

1. 一个页面只有一个 `<h1>`：用于主要标题
2. 不要跳过标题级别：按顺序使用（如 h1 → h2 → h3，不要 h1 → h3）
3. 保持层次清晰：像写文档大纲一样思考

### 和普通字体的视觉对比

| 元素 | 默认字体大小 (相对值) | 默认字体大小 (像素近似值) | 与普通字体(16px)对比 |
| -- | -- | -- | -- |
| `<h1>` | 2em | 32px | 比普通字体大很多 (2倍) |
| `<h2>` | 1.5em | 24px | 比普通字体大 |
| `<h3>` | 1.17em | 约18.72px | 比普通字体稍大 |
| `<h4>` | 1em | 16px | 与普通字体相同 |
| `<p>` (普通字体) | 1em | 16px | 基准大小 |
| `<h5>` | 0.83em | 约13.28px | 比普通字体小 |
| `<h6>` | 0.67em | 约10.72px | 比普通字体小很多 |

## 段落 `<p>`

`<p>` 标签用于定义文本段落。

```html
<p>这是一个段落。段落会自动在前后创建一些空白边距，使内容更易读。</p>
<p>这是另一个段落。浏览器会自动在每个段落之间添加适当的间距。</p>
```
