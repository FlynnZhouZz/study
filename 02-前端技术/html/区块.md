# 区块

HTML 区块元素用于组织和分组内容，创建文档结构。

## 主要区块元素

| 元素 | 用途 | 说明 |
| -- | -- | -- |
| `<div>` | 通用容器 | 无语义，用于样式和布局 |
| `<section>` | 内容分组 | 有主题的独立内容区块 |
| `<article>` | 独立内容 | 可独立分发或重复使用的内容 |
| `<header>` | 页眉 | 页面或区块的引导内容 |
| `<footer>` | 页脚 | 页面或区块的结尾内容 |
| `<nav>` | 导航 | 主要导航链接集合 |
| `<aside>` | 侧边内容 | 与主要内容相关的附加内容 |
| `<main>` | 主要内容 | 文档的独特核心内容 |

## 最佳实践

1. 语义化优先 - 使用合适的语义化标签
2. 避免过度使用div - 只在没有合适语义标签时使用
3. 合理嵌套 - 保持清晰的文档结构
4. 可访问性 - 使用地标角色(role)增强可访问性
5. SEO优化 - 语义化标签有助于搜索引擎理解内容
