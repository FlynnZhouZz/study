# 页面结构化和语义化

## 结构化

指使用适当的HTML标签来搭建页面的骨架和布局，将内容划分为有逻辑的、可识别的部分。 <br>
实现方式：传统上用 `<div>` 配合CSS，现代则更多使用HTML5的语义化标签。

> 类比：就像盖房子时的建筑设计图，它定义了哪里是承重墙、哪里是客厅、哪里是卧室。它关注的是“块”与“布局”。

## 语义化

指为页面中的内容块选择合适的HTML标签，使得标签本身就能传达其承载内容的含义和角色。 <br>
实现方式：使用具有明确含义的HTML标签，如 `<header>`, `<article>`, `<h1>-<h6>`, `<nav>`, `<strong>` 等，而不是通用的 `<div>` 和 `<span>`。

> 类比：在建筑图纸上，不仅画出房间，还要标上“卧室”、“厨房”、“卫生间”的标签。它关注的是“这个部分是什么”。

## 结构化和语义化对比

| 维度 | 结构化 | 语义化 |
| -- | -- | -- |
| 核心关注点 | 布局、容器、内容块的组织 | 含义、角色、内容的本质 |
| 要解决的问题 | “内容如何分组和排列？” | “这个内容块代表什么？” |
| 实现方式 | 使用任何HTML标签创建层次和分区（包括`<div>`） | 使用具有明确含义的HTML标签 |
| 机器理解程度 | 知道有内容块，但不知道其作用 | 知道内容块的角色和重要性 |
| 类比 | 建筑的承重墙和房间隔断 | 房间门牌上的“会议室”、“休息室” |

## 从代码角度理解

只有结构化，没有语义化
```html
<!-- 这个页面有结构，但没有语义 -->
<div class="top-area">
  <div class="logo">网站名</div>
  <div class="menu">
    <a href="#">首页</a>
    <a href="#">关于</a>
  </div>
</div>
<div class="content-area">
  <div class="main">
    <div class="big-title">文章标题</div>
    <div class="text">文章内容...</div>
  </div>
  <div class="sidebar">
    <div class="small-title">相关链接</div>
    <!-- 侧边栏内容 -->
  </div>
</div>
<div class="bottom-area">版权信息</div>
```
分析：页面有清晰的结构（顶部、内容区、底部），但机器看到的全是`<div>`，无法理解每个部分的角色。

结构化 + 语义化
```html
<!-- 这个页面既有结构，又有语义 -->
<header>
  <h1>网站名</h1>
  <nav>
    <a href="#">首页</a>
    <a href="#">关于</a>
  </nav>
</header>
<main>
  <article>
    <h2>文章标题</h2>
    <p>文章内容...</p>
  </article>
  <aside>
    <h3>相关链接</h3>
    <!-- 侧边栏内容 -->
  </aside>
</main>
<footer>版权信息</footer>
```
分析：页面不仅结构清晰，而且每个部分都有明确的语义含义。

## 为什么它们如此重要？

1. 对SEO（搜索引擎优化）至关重要

- 帮助搜索引擎理解内容：搜索引擎爬虫会根据语义化标签来理解页面内容的结构、层次和关键部分。例如，它会知道 `<main>` 里的内容是核心， `<nav>` 里的内容是导航链接。
- 提升关键词权重：在 `<h1>`、 `<title>` 等标签中的关键词通常被认为比普通 `<p>` 中的关键词更重要。
- 改善富媒体搜索结果：良好的结构有助于搜索引擎生成更丰富的搜索结果摘要（如面包屑导航）。

2. 极大的可访问性优势

- 屏幕阅读器友好：视障用户依赖屏幕阅读器。语义化标签为屏幕阅读器提供了“路标”，用户可以快速在标题、导航、主内容区之间跳转，而不是线性地听完所有 `<div>`。
- 键盘导航支持：一些语义化元素（如 `<nav>`）本身就带有可访问性功能，方便键盘用户操作。

3. 对开发者更友好

- 代码可读性和可维护性：查看 `<article>` 标签比查看 `<div class="post">` 更能直观地理解其内容。新团队成员能更快上手。
- 开发效率：清晰的结构减少了寻找和修改特定功能区域的成本。

4. 对未来兼容

- 遵循标准的代码更有可能在未来浏览器和设备上正常工作。
