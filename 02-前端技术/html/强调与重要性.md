# 强调与重要性

## 强调元素

1. `<em>` - 强调（斜体）
- 语义：表示语音重读的强调
- 默认样式：斜体
- 屏幕阅读器：可能会改变语调来朗读

2. `<strong>` - 强烈强调（加粗）
- 语义：表示内容的重要性、严重性或紧迫性
- 默认样式：加粗
- 屏幕阅读器：可能会用更强的语气朗读

3. `<mark>` - 高亮标记
4. `<i>` - 技术术语、外来语等
5. `<b>` - 引起注意但不增加重要性

## 语义差异总结

| 标签 | 语义含义 | 使用场景 | 默认样式 |
| -- | -- | -- | -- |
| `<em>` | 强调 | 语音重读、轻微强调 | 斜体 |
| `<strong>` | 强烈重要性 | 重要内容、警告、关键信息 | 加粗 |
| `<i> `| 不同语气 | 技术术语、外来语、想法 | 斜体 |
| `<b>` | 引起注意 | 关键词、产品名称 | 加粗 |
| `<mark>` | 突出显示 | 搜索结果、重点标记 | 黄色背景 |

## 最佳实践

```html
<!-- 根据语义选择标签 -->
<p>这个操作<strong>不可撤销</strong>，请<em>谨慎</em>执行。</p>

<!-- 嵌套使用 -->
<p><strong>重要：请<em>立即</em>更新您的密码</strong></p>

<!-- 结合CSS -->
<strong class="warning-text">危险操作！</strong>
```
