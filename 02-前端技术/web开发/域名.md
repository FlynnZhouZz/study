# 域名

## 简述

域名就像是互联网世界的“通讯录”或“地址簿”。

它让我们人类能够用容易记忆的单词（如 google.com）来访问网站，而无需记住那些冰冷、复杂的数字串（如 142.251.42.206）。

## 概述

域名是网站在互联网上的唯一名称。它是由一串用点分隔的字符组成，用于在数据传输时标识计算机的电子方位（有时也指地理位置）。

人类友好 vs. 机器友好：

IP地址：是计算机在互联网上的真实地址，由一长串数字组成（例如 IPv4: 192.0.2.1，或更长的 IPv6: 2001:0db8:85a3:0000:0000:8a2e:0370:7334）。这对机器很友好，但人类难以记忆。 <br>
域名：将难记的IP地址转换为像 baidu.com、wikipedia.org 这样易于理解和记忆的名字。

## 域名的结构：像是一个倒置的树

一个完整的域名从右到左，层级越来越高。以 www.example.com 为例：

根域：最顶级的，通常省略不写，用一个点 . 表示。

顶级域：也叫域名后缀，是最后一部分，如 .com。
- 通用顶级域：如 .com（商业机构）、.org（非营利组织）、.net（网络公司）、.edu（教育机构）等。
- 国家和地区代码顶级域：如 .cn（中国）、.uk（英国）、.jp（日本）等。

二级域：这是域名的核心部分，是你自定义和注册的独特标识。例如 example。

子域：也叫主机名，是二级域之前的部分，最常见的是 www。你可以创建任意子域，如 mail.example.com、shop.example.com、docs.example.com 等。

## 域名是如何工作的？DNS 系统

域名本身只是一个“名字”，它必须指向一个真实的IP地址才能访问网站。这个转换工作是由 DNS 来完成的。 <br>
DNS 就像是互联网的“巨型电话簿”或“导航系统”。

当你访问 google.com 时，会发生以下过程：
1. 你在浏览器中输入 google.com 并按下回车。
2. 你的计算机会向 DNS 解析器（通常由你的网络服务商提供）提问：“google.com 的IP地址是什么？”
3. DNS 解析器会去查询全球分布式的 DNS 根服务器、TLD 服务器（负责 .com 的服务器），最终找到负责 google.com 的 权威名称服务器。
4. 权威名称服务器告诉解析器 google.com 对应的IP地址是 142.251.42.206。
5. 解析器将这个IP地址返回给你的浏览器。
6. 浏览器根据这个IP地址，连接到谷歌的服务器，从而加载出谷歌的首页。

整个查询过程在毫秒之间完成，用户完全无感。

## 为什么域名如此重要？

1. 品牌标识：域名是你的在线品牌和身份。apple.com 就是苹果公司在线上的门面。
2. 可访问性：没有域名，用户就需要记住复杂的IP地址才能访问网站，这几乎是不可能的。
3. 专业性：拥有一个自定义的域名（如 yourcompany.com）比使用第三方提供的子域（如 yourcompany.wordpress.com）看起来专业得多。
4. 电子邮件地址：域名也用于创建专业的电子邮件地址，如 name@yourcompany.com。
5. 数字资产：好的域名具有很高的价值，被视为一种重要的数字资产，可以被买卖和投资。

## 如何获取一个域名？

域名需要通过 域名注册商 进行注册和按年付费。全球知名的注册商有 GoDaddy、Namecheap 等，国内有阿里云（万网）、腾讯云等。
