# 命令行

## 终端

终端是一个文本界面，用于执行基于文本的程序。<br>
如果你正在运行任何用于 web 开发的工具，那么几乎可以保证你必须打开命令行并运行一些命令来使用你所选择的工具 (你经常会看到这样的工具被称为CLI 工具命令行接口工具)。

## 调起终端

### 1、Windows系统

Windows 主要有两个命令行工具：命令提示符 (CMD) 和 PowerShell。从 Windows 10 开始，微软还推出了更现代化的 Windows Terminal。

方法一：<br>
1. `Win+R`，打开“运行”对话框
2. 输入：
    - cmd: 命令提示符
    - PowerShell: 启动 PowerShell
    - wt: 启动 Windows Terminal (如果已安装)

方法二：<br>
1. 点击开始按钮或按下 Win 键。
2. 直接输入 cmd、powershell 或 终端，然后按回车即可。

### 2、 macOS 系统

macOS 的终端应用叫做 Terminal，或者你也可以使用功能更强大的替代品，如 iTerm2。

方法一：使用聚焦搜索 (最推荐) <br>
1. 按下 Cmd + Space 组合键，打开聚焦搜索。
2. 输入 Terminal 或 终端，然后按回车。

方法二：通过启动台 <br>
1. 打开“启动台”（可以在程序坞点击，或使用 F4 键/触控板四指捏合手势）。
2. 找到“其他”文件夹，里面就有 Terminal 应用。

方法三：通过访达 <br>
1. 打开“访达”。
2. 进入“应用程序” -> “实用工具”文件夹，找到并双击 Terminal。

### 3、Linux 系统

Linux 的桌面环境众多（如 GNOME, KDE, XFCE等），但调起终端的方法大同小异。

方法一：使用快捷键 (最通用) <br>
1. 在绝大多数 Linux 发行版和桌面环境中，默认的打开终端的快捷键是：`Ctrl + Alt + T`
2. 这是首选的、最快的方法。

方法二：通过应用程序菜单 <br>
1. 点击屏幕左上角或左下角的“应用程序”菜单。
2. 在系统工具或工具分类下，找到名为 Terminal、Konsole (KDE)、GNOME Terminal 等的应用。

方法三：使用应用启动器 <br>
1. 按下 Alt + F2（在某些桌面环境中，如 GNOME）。
2. 输入 gnome-terminal、konsole 或 xfce4-terminal（取决于你的桌面环境），然后按回车。

方法四：右键菜单 <br>
在桌面或文件管理器的空白处点击右键，通常会有 “在终端中打开” 或类似的选项。

## 基本的内置终端命令

### 导航计算机的文件系统以及基本级别的任务，如创建、复制、重命名和删除：

- cd: 移动你的目录结构
- mkdir: 建立目录
- touch: 创建文件（修改他们的原数据）
- cp: 复制文件
- mv: 移动文件
- rm: 删除文件或目录
- ls: 列出目录内容

### 介绍命令选项

- -l

如：
```bash
ls -l
```

### 下载在特定的 url 找到的文件

- curl

### 在较大的文件体中寻找特定的片段

- grep

### 主页查看文件的内容

- less
- cat

### 操作和转换文本流（例如，将 HTML 文件中 <div> 的所有实例改为 <article>）

- awk
- tr
- sed
