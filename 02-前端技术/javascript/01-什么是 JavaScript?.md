# 什么是 JavaScript?

JavaScript（简称 JS） 是一种高级的、解释型的编程语言（运行在浏览器和服务器上的编程语言），主要用于网页开发，为网页添加交互功能。它是现代 Web 开发的三大核心技术之一，与 HTML（结构） 和 CSS（样式） 共同构建了互联网的基础。

## 主要特性

1. 脚本语言

-   直接在浏览器中运行，无需编译。
-   通过 `<script>` 标签嵌入 `HTML` 或引用外部 `.js` 文件。

2. 事件驱动与交互性

-   响应用户操作（如点击、输入、滚动等）。
-   动态修改网页内容、样式，实现动画、表单验证、弹窗等。

3. 跨平台与浏览器支持

-   所有现代浏览器均支持 JavaScript（Chrome、Firefox、Safari 等）。
-   可通过 Node.js 在服务器端运行，实现全栈开发。

4. 动态类型与灵活性

-   变量类型无需预先声明，运行时自动推断。
-   支持面向对象、函数式等多种编程范式。

## JavaScript 的核心组成

-   ECMAScript：语言本身（语法、变量、函数、Promise）
-   DOM：操作网页结构
-   BOM：操作浏览器（window、location）

## 核心用途

-   前端开发：操作 DOM、处理事件、发起异步请求（AJAX/Fetch）。
-   后端开发：通过 Node.js 构建服务器、API 和工具。
-   移动/桌面应用：使用 React Native、Electron 等框架。
-   游戏开发：结合 HTML5 Canvas 或 WebGL。
-   物联网（IoT）：运行在嵌入式设备上。
-   小程序、插件、脚本工具

## 生态系统

-   框架与库：React、Vue.js、Angular（前端）；Express.js（后端）。
-   工具链：npm/yarn（包管理）、Webpack（打包）、Babel（转译）。
-   标准演进：ES6（ES2015）及后续版本引入了类、模块、箭头函数等现代语法。

## JavaScript 和 Java 的区别

> 名字像，但完全不是一回事

| 对比项   | JavaScript       | Java       |
| -------- | ---------------- | ---------- |
| 类型     | 动态类型         | 静态类型   |
| 运行环境 | 浏览器 / Node.js | JVM        |
| 用途     | Web、App、脚本   | 企业级后端 |
| 关系     | ❌ 无直接关系    | ❌         |
