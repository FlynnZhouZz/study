## 软件准备

### 步骤1、清理不必要的软件

- 卸载安全杀毒软件 - 如：360安全卫士、电脑管家、金山毒霸等，完全没必要，系统自带的即可。他们只是给纯电脑小白使用的，你作为相关专业人员，就别让自己显得“外行”了。还只会消耗紧张的内存、显卡。
- 卸载游戏 - 如果是Windows，一定要卸载游戏。可以另外配一台电脑来打游戏，不要在工作机上玩游戏。在说程序员知识每天都在更新，你学都学不过来，哪还有时间去打游戏呀。

### Git配置

配置用户名和邮箱
```bash
git config --global user.name "你的姓名"
git config --global user.email "你的公司邮箱或个人邮箱"

# 检查
# 查看所有全局配置项
git config --global --list
# 查看所有（全局+本地）配置项
git config --list
```

PS: 邮箱最好与你的代码托管平台（GitHub, GitLab, Gitee）的注册邮箱一致，这样平台才能正确关联你的提交

SSH 公钥：
```bash
# 生成SSH公钥
ssh-keygen -t ed25519 -C "备注信息"

# 输出如下  中间通过三次回车键确定

Your public key has been saved in /home/git/.ssh/id_ed25519.pub
The key fingerprint is:
SHA256:ohDd0OK5WG2dx4gST/j35HjvlJlGHvihyY+Msl6IC8I Gitee SSH Key
The key's randomart image is:
+--[ED25519 256]--+
|    .o           |

|  ...O.o +       |

|. ...o o..+* *   |
|.E. o . ..+.O    |

# 查看生成的 SSH 公钥和私钥：
ls ~/.ssh/

# 读取公钥文件 ~/.ssh/id_ed25519.pub：
cat ~/.ssh/id_ed25519.pub
```
- -t key 类型
- -C 注释

读取到公钥，将字符复制粘贴到对应git平台（如：github、gitee）等，配置好本机就可以有权限拉取代码了。
