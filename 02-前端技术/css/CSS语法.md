# CSS 语法

## CSS 基本语法结构

```css
选择器 {
    属性: 值;
}
```

解释：

-   `p` → 选择器（selector），告诉浏览器要选中 `<p>` 元素
-   `color` → 属性（property）
-   `red` → 值（value）
-   `;` → 多条规则之间用分号分隔
-   `属性: 值;` → 声明

## CSS 注释

注释不会被浏览器解析。

```css
/* 这是一个单行注释 */

/*
   这是一个
   多行注释
*/

p {
    color: blue; /* 将段落文字颜色设置为蓝色 */
    /* font-size: 14px; 这行代码被注释掉了，不会生效 */
}
```

## 大小写（Case）

CSS 大小写不敏感：

```css
color
COLOR
Color
```
都可以，但标准写法用小写。

## 单位（Units）

CSS 常用单位：
- 长度：px、em、rem、vw、vh
- 百分比：%
- 颜色：red、#333、rgb()、hsl()

## 值类型（Value Types）

类型示例：
- 数字：1, 0.5
- 字符串："hello"
- 关键词：auto, none, block
- 函数：url(), rgb(), calc()
- 多值：margin: 10px 20px;

## 结尾分号是否必须？

- 一条声明 → 通常需要 ;
- 最后一条声明 → 可以不加，但建议加，避免出错

## CSS 代码格式规范

1. 每行只写一个声明
2. 使用缩进：通常使用 2 个或 4 个空格来缩进声明块内的代码
3. 使用小写：虽然 CSS 不区分大小写，但属性和值通常使用小写字母（除了字体名称和特定值）。
