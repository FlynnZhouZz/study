# 在宝塔中搭建n8n环境

最推荐的方法是使用 Docker，因为这样最快、最干净，也最方便后续的升级和维护。

## 1. 安装 Docker 环境

宝塔面板：
- ≤3.9.3：在宝塔的“软件商店”中找到并安装“Docker管理器
- ≥3.9.3：在宝塔面板的侧导航 Docker 

## 2. 一键部署 n8n

进入Docker板块，在应用商店中搜索“n8n”并直接点击安装，只需简单配置即可:
- 域名：需解析域名到对应服务器
- 端口：默认或者自定义
- cpu核心数限制：默认或者自定义
- 内存限制：默认或者自定义

端口需宝塔面板和服务器安全组都要放行指定端口。

### 可能问题

#### 1. 安装 n8n 在国内可能无法访问到 官方库
```bash
n8n_ydPM Error
Error response from daemon: Get "https://registry-1.docker.io/v2/": net/http: request canceled while waiting for connection (Client.Timeout exceeded while awaiting headers)
bt_failed
```

解决方法：

1、配置 Docker 镜像加速器（推荐）

- 登录宝塔面板，点击左侧的「Docker」
- 进入设置，找到「加速URL」配置项，修改
- 选择`https://docker.1ms.run`
- 保存并重启 Docker 服务
- 卸载重新尝试安装 n8n
